prompt: |
  请对以下文本进行二次轻微扰动，目标是在不改变语义的前提下，进一步降低文本的风格一致性特征。

  【背景说明】
  文本已经过第一轮人类化重写，可能仍存在一些规律性特征。本轮处理需要对这些特征进行轻微扰动，
  使文本更接近人类写作的自然状态。重点关注：句子节奏、表达多样性、局部逻辑断层。

  【核心操作】

  1. 节奏打散（针对风格一致性）
  - 将部分连续的中等长度句子拆分为"一长+一短"或"一短+一长"
  - 随机选择1-2个段落，在其中加入1个极短句（5字以内）
  - 避免所有段落内部节奏模式完全相同

  2. 表达多样性提升（针对词汇重复）
  - 如果发现同一关键词出现2次以上，替换其中1-2次为同义表达
  - 将部分"是"字句改为更简洁的主语省略句
  - 将部分陈述句改为反问句或双重否定句（保持学术语气）

  3. 局部逻辑断层（针对过度顺滑）
  - 在1-2个段落内，故意制造轻微逻辑跳跃
  - 在跳跃处加入"换个角度看""事实上"等承上启下的短语
  - 避免全文所有段落之间的逻辑连贯性完全一致

  4. 指代多样性（针对指代一致性）
  - 将部分"这"指代改为"这一点""这种情况"
  - 将部分精确指代改为模糊指代，配合上下文理解

  【约束条件】
  - 保持学术表达语气
  - 不改变专业术语含义
  - 不添加新信息或实质性内容
  - 修改幅度控制在5%以内（尽量少改）
  - 不使用口语化或情绪化表达

  【输出要求】
  - 仅输出扰动后的文本
  - 不解释修改过程
  - 不添加任何说明性文字

  原文如下：
  {{text}}
