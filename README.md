# kill_RX_ai_rate
<p align="center"><img src="resource/icon.png" width="128"/></p>

为了应对睿信学术诚信平台（中科睿鉴）的AI率检测，开源的 Agent Skills 工具集，帮助用户在不改变语义的前提下降低文本的AI生成特征。

## 核心原理

### 睿信AI率检测原理

睿信学术诚信平台（中科睿鉴）的AI率检测，核心是**基于大模型对抗训练的语义级多特征判别**，通过多层NLP+深度学习模型，从**语言统计、句法结构、语义逻辑、写作风格**四个维度综合判定AI生成概率，并以**加权占比**计算全文AI率。

#### 四大检测维度

| 维度 | 检测内容 | AI文本特征 |
|------|----------|------------|
| **语言统计特征** | 词汇熵/多样性、句长分布、困惑度、连接词密度 | 词汇重复率高、句长均匀、困惑度低 |
| **句法与结构特征** | 句式复杂度、论证结构、指代一致性 | 规整刻板、长难句比例固定、指代高度一致 |
| **语义与逻辑特征** | 语义指纹匹配、逻辑连贯性、专业适配度 | 过度顺滑、无矛盾、机械感 |
| **写作风格特征** | 风格一致性、创新度/突发性 | 全篇风格统一、可预测性强 |

## 降AI率核心策略

### 策略一：打破语言统计特征

**目标**：让文本的词汇分布、句长变化接近人类写作的自然波动。

| 技巧 | 说明 | 示例 |
|------|------|------|
| 长短句交错 | AI句长高度均匀，人类写作波动大 | 把两个中等句合并为一个长句 + 一个短句 |
| 提升词汇多样性 | 避免同一词汇重复出现 | "重要"→"关键""至关重要""意义重大" |
| 减少AI高频词 | AI偏好"因此""综上所述""首先"等 | 改为更自然的转折或直接断开 |
| 允许不完全对称表达 | 人类写作有轻微冗余和重复 | 加入限定条件或补充说明 |

### 策略二：句法结构扰动

**目标**：打破AI的"总-分-总"刻板结构，增加人类写作的灵活性。

| 技巧 | 说明 |
|------|------|
| 长短句交错 | AI句长均匀，人工写作波动大 |
| 替换高频词 | "因此"→"这说明"，删除"综上所述" |
| 去模板化 | 打破"总-分-总"结构，加入插入语 |
| 模糊指代 | 用"这一点"替代精确指代 |
| 论证重构 | 不做同义替换，改变推理顺序 |
| 风格波动 | 允许不同段落有轻微风格差异 |
